# 环境
  * OS: Centos8.0 bit64
  * neovim: NVIM v0.9.4
  * tmux: tmux 2.7
  * python3.8

# 快速安装
  * curl https://raw.githubusercontent.com/kindf/dotfiles/main/bootstrap.sh | bash

# tmux快捷键
  * ctrl + e + i：竖分屏
  * ctrl + e + o：横分屏
  * crtl + e + z：当前窗口最大化
  * crtl + e + n：向前切换页签
  * crtl + e + m：向后切换页签
  * crtl + e + h：向左切换窗口
  * crtl + e + j：向上切换窗口
  * crtl + e + k：向下切换窗口

# 插件及快捷键
  * leader 键为 ","
  * 目录树插件 nerdtree
    * \<leader\>t：打开目录树
  * 代码注释插件 vim-commentary
    * 选中行，/进行注释
  * 快速跳转插件 vim-easymotion
    * J：快速跳转
  * 文件搜索，模糊搜索插件 leaderF
    * \<C-p\>：模糊搜索文件
    * \<C-n\>：文件函数搜索
    * \<C-m\>：打开最近访问文件列表
    * 选中单词，\<leader\>g：全局搜索选中单词
    * \<leader\>f：全局搜索
  * 有道词典翻译插件 vim-youdao-translater
    * \<C-t\>：翻译
  * 命令行主题插件 promptline.vim
  * tmux状态栏插件 tmuxline.vim
  * 启动界面插件 vim-startify
  * 主题插件 nerodark.vim
  * lsp
    * gd ：查看定义
    * gr ：查看引用
    * go ：打开代码预览
    * \<C-t\> ： 开关 nvim 内终端
